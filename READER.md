# GamePrice Watcher Bot 🎮📜

> Автоматизированный агрегатор и мониторинг кроссплатформенного игрового контента.

Telegram-бот на Python для поиска цен на компьютерные игры, сравнения площадок (Steam, CheapShark, Plati, Kupikod, Steambuy, GGSEL) и автоматического уведомления пользователя при падении цены ниже заданного порога в рублях.

## 🔥 Ключевые возможности
- **Поиск цен:** параллельный опрос магазинов и агрегаторов с автоматической конвертацией USD ➡️ ₽ по курсу ЦБ РФ.
- **Wishlist:** персональный список отслеживания. Планировщик проверяет цены каждые 2 часа и отправляет push-уведомление при достижении лимита.
- **История цен:** генерация линейных графиков изменения стоимости через `matplotlib`.
- **Ловец халявы:** автоматический мониторинг бесплатных раздач на крупных площадках.
- **Мультиязычность:** полная локализация интерфейса (RU / EN).

## 🛠 Технологический стек
- **Язык:** Python 3
- **Архитектура:** Асинхронная (`asyncio`, `aiohttp`)
- **Telegram API:** `aiogram 3`
- **База данных:** `SQLite` (`aiosqlite`)
- **Планировщик задач:** `APScheduler`
- **Визуализация:** `matplotlib`
