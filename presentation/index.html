Актуальность и проблема

  • Цены разбросаны: Steam, ключи, российские магазины
  • Разные валюты и региональные ограничения (РФ / СНГ)
  • Нет единого удобного инструмента в привычном мессенджере

Идея: один бот — поиск, сравнение, уведомления

Цель и задачи

Цель: агрегировать цены и уведомлять о снижении

  • Поиск игры по названию, сокращениям и ссылке Steam
  • Сбор цен из нескольких API и веб-источников
  • Wishlist с порогом в рублях и фоновой проверкой
  • История цен и построение графика
  • Интерфейс на русском и английском языках

Источники данных

  • CheapShark API — ключи, USD → ₽ (курс ЦБ РФ)
  • Steam Store API — цена для региона RU или US
  • Plati.market — официальный XML API
  • Kupikod, Steambuy, GGSEL — парсинг HTML-страниц

Минимальная цена = минимум по всем источникам в рублях

Функция: поиск (/search)

  • Алиасы («гта 5») + нечёткий поиск CheapShark / Steam
  • Параллельный сбор цен (asyncio.gather)
  • Сортировка предложений от дешёвого к дорогому
  • Вывод: «от X ₽», магазины, DLC, AppID, CheapShark

Wishlist и планировщик

/add — ссылка на игру + порог в ₽

ЗадачаПериод

Бесплатные игры (Epic, Steam)раз в 24 часа

При цена ≤ порог → уведомление в Telegram

База данных (SQLite)

  • users — язык, telegram_id
  • tracked_games — wishlist, порог, last_price
  • price_history — точки для графика
  • favorites — патчи по AppID
  • freebie_cache — антиспам раздач

Файл: gameprice.db · параметризованные SQL-запросы
