GamePrice Watcher — презентация для защиты
==========================================

1) ПОКАЗ В БРАУЗЕРЕ (рекомендуется)
   Дважды щёлкни: presentation\index.html
   Или: ПКМ → Открыть с помощью → Chrome / Edge

   Управление:
   - Стрелки ← → или пробел — следующий/предыдущий слайд
   - F11 — полный экран
   - Клик мышью: правая половина экрана — вперёд, левая — назад

2) СОХРАНИТЬ В PDF (для сдачи)
   Открой index.html → Ctrl+P → «Сохранить как PDF»
   Включи «Фоновая графика»

3) ПЕРЕД ЗАЩИТОЙ
   Замени на слайде 1: [ФИО], [№ группы], [руководитель], [год]

4) PowerPoint (.pptx)
   Если установлен Python:
   pip install python-pptx
   python build_pptx.py
   Файл появится: GamePrice_Watcher.pptx
